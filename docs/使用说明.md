# FitVision 使用说明

## 📌 项目简介

FitVision 是一个基于 Django + Vue.js 的 AI 健身教练平台，提供运动追踪、姿态诊断、个性化推荐等功能。

---

## 🚀 启动项目

### 🎯 快速开始（首次使用）

**第一步：确保服务运行**
- PostgreSQL 数据库已启动（端口 5432）
- 前后端服务器已启动（参考下方步骤）

**第二步：注册并登录**
1. 打开浏览器访问：http://localhost:5173/
2. 点击"立即注册"创建账号
3. 或者创建管理员账号（推荐）：
   ```powershell
   cd G:\FitVision\backend
   conda activate fitvision
   python manage.py createsuperuser
   ```
4. 使用账号登录即可开始使用

---

### 前置条件

确保以下服务已启动：
- ✅ **PostgreSQL** 数据库服务（端口：5432）
- ✅ **Redis** 服务（端口：6379，如使用缓存/任务队列）

---

### 1️⃣ 启动后端服务器

打开 **PowerShell** 终端，执行以下命令：

```powershell
# 进入后端目录
cd G:\FitVision\backend

# 激活 conda 虚拟环境
conda activate fitvision

# 启动 Django 开发服务器
python manage.py runserver
```

**启动成功标志：**
```
Starting development server at http://127.0.0.1:8000/
```

后端地址：**http://127.0.0.1:8000/**

---

### 2️⃣ 启动前端服务器

打开**另一个** PowerShell 终端，执行以下命令：

```powershell
# 进入前端目录
cd G:\FitVision\frontend

# 启动 Vite 开发服务器
npm run dev
```

**启动成功标志：**
```
➜  Local:   http://localhost:5173/
```

前端地址：**http://localhost:5173/**

## � 用户登录和注册

### 方式一：使用前端界面（推荐）

1. **访问前端应用**：http://localhost:5173/
2. 首次使用需要注册账号：
   - 点击"立即注册"链接
   - 填写用户名、邮箱和密码
   - 完成注册后自动跳转登录

3. **登录账号**：
   - 访问登录页面：http://localhost:5173/login
   - 输入用户名和密码
   - 登录成功后会跳转到主页

### 方式二：创建管理员账户

如果需要访问后台管理界面，需要创建超级管理员：

```powershell
# 进入后端目录
cd G:\FitVision\backend

# 激活环境
conda activate fitvision

# 创建超级管理员
python manage.py createsuperuser
```

按提示输入：
- **用户名**：例如 `admin`
- **邮箱**：例如 `admin@example.com`
- **密码**：至少8位（输入时不显示）

创建成功后可以访问：
- **后台管理**：http://127.0.0.1:8000/admin/
- **前端登录**：http://localhost:5173/login （使用相同账号）

### 方式三：使用 API 直接注册

使用 Postman 或 curl 调用注册接口：

```bash
# 注册新用户
curl -X POST http://127.0.0.1:8000/api/auth/register/ \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "email": "test@example.com",
    "password": "password123"
  }'

# 登录获取 Token
curl -X POST http://127.0.0.1:8000/api/auth/login/ \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "password": "password123"
  }'
```

### 已有功能

✅ **用户注册** - `/api/auth/register/`
✅ **用户登录** - `/api/auth/login/` （返回 JWT Token）
✅ **个人资料** - `/api/auth/profile/`
✅ **用户信息** - `/api/auth/me/`
✅ **训练记录** - `/api/auth/training/logs/`
✅ **目标管理** - `/api/auth/goals/`
✅ **数据统计** - `/api/auth/stats/`

## 🔧 常用命令

### 数据库相关

```powershell
# 进入后端目录并激活环境
cd G:\FitVision\backend
conda activate fitvision

# 创建迁移文件（修改模型后）
python manage.py makemigrations

# 执行数据库迁移
python manage.py migrate

# 创建超级管理员账户
python manage.py createsuperuser

# 重置数据库（谨慎使用）
python manage.py flush
```

### 依赖管理

```powershell
# 后端：安装新的 Python 包
cd G:\FitVision\backend
conda activate fitvision
pip install <package_name>
pip freeze > requirements.txt  # 更新依赖列表

# 前端：安装新的 npm 包
cd G:\FitVision\frontend
npm install <package_name>
npm install  # 安装所有依赖
```

### 其他常用命令

```powershell
# 后端：运行测试
cd G:\FitVision\backend
conda activate fitvision
python manage.py test

# 后端：Django Shell（调试用）
python manage.py shell

# 前端：构建生产版本
cd G:\FitVision\frontend
npm run build
```

## 🐛 常见问题

### 1. 数据库连接失败

**错误信息：** `database "fitvision" does not exist`

**解决方法：**
```sql
-- 使用 psql 或 pgAdmin 创建数据库
CREATE DATABASE fitvision;
```

### 2. 端口被占用

**错误信息：** `Error: That port is already in use`

**解决方法：**
```powershell
# 查找占用端口的进程
netstat -ano | findstr :8000
netstat -ano | findstr :5173

# 结束进程（PID 是上一步找到的进程号）
taskkill /PID <PID> /F
```

### 3. Python 模块未找到

**错误信息：** `ModuleNotFoundError: No module named 'xxx'`

**解决方法：**
```powershell
cd G:\FitVision\backend
conda activate fitvision
pip install -r requirements.txt
```

### 4. npm 依赖问题

**解决方法：**
```powershell
cd G:\FitVision\frontend
rm -r node_modules
rm package-lock.json
npm install
```
